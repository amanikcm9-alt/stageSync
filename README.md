# StageSync

## Description

StageSync est une plateforme web de gestion des stages développée dans le cadre de mon Projet de Fin d'Études.

Cette plateforme permet de gérer :

- Les stagiaires
- Les candidatures
- Les encadrants
- Les ressources humaines
- Les administrateurs
- Les évaluations
- Les tableaux de bord

---

## Technologies utilisées

- Laravel
- PHP
- JavaScript
- Bootstrap
- MySQL
- HTML
- CSS
- Git

---

## Fonctionnalités

- Authentification
- Gestion des rôles
- Gestion des utilisateurs
- Gestion des candidatures
- Gestion des stages
- Tableau de bord
- Notifications

---

## Installation

```bash
git clone https://github.com/amanikcm9-alt/stageSync.git

cd stageSync

composer install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
